SCP-2744
rating: -16+–x

SCP-2744 prior to containment

Item #: 2783

Object Class: Euclid

Special Containment Procedures:
SCP-2744 is contained in a locker in a large room, and
is to have two armed guards blocking
the entrance to the room at all times.
Any entities or objects from
SCP-2744 are to be detained or executed immediately.

Description: SCP-2744 appears to be a blue, cat face with a partly erased brown face made out of memory foam. It is 3 Inches long in width and two inches in height. When SCP-2744 is squished a randomized event will happen while the subject passes out it is presumed to be SCP-2744-2, The event happens close by or on the subject. SCP-2744's events are unpredictable as it is never what the subject predicts, It is guessed that it can sense what the subject is predicting. Events SCP-2744 makes can be dangerous, and wonderful. One recorded case of Agent Grover squishing SCP-2744, resulted in Agent Grover passed out with an unknown entity appearing nearby, The entity was detained as soon as possible.
SCP-2744-2 is presumed to be an alternative universe. Subjects claim that it is a corrupted universe of ours. When the event is happening the subject usually passes out for one hour.

Interviewed: [Agent Grover]

Interviewer: [Agent ████ ]

Foreword: [Agent Grover passed out when squishing SCP-2744, he was immediately detained, When Agent Grover woke up he was interviewed by Agent ████]

<Begin Log>

Interviewer: [Hello, Agent Grover I will be interviewing you.]

Agent Grover: [Why?]
Interviewer: [When you squished SCP-2744 you passed out, Did you feel anything?]

Agent Grover: [No, Everything changed.]
Interviewer: [What do you mean by this?]

Agent Grover: [When everything changed, Everything looked broken, corrupted.]
Interviewer: [What did you see in this broken universe?]

Agent Grover: [I saw █████, Corpses of the ████████████████]

<End Log>

Closing Statement: [The recording cut off, Agent Grover and Agent ████ disappeared, it is unknown where they are. The ███████ said it all went black the second they disappeared.]
